JS Fundamentals

A good way to understand JavaScript is to first explain the basics of a website. Websites are essentially are tool to open and view simple text files. When we use a text file for a website, we will use the .HTML format. HTML files alone are a bit booring to look at as they contain simple text. We can make them more interesting by adding colors, fonts, borders etc. We can call this styling and is best done by linking a .CSS file to the original .HTML file we have used for our website. The .CSS file will contain special code designed to add style attributes to the text in the .HTML file.

If we want to add interactivity or dynamic elements to a website, we need a more powerful tool. One of these tools is called Javascript. JavaScript works the same way as a .CSS files does. All we need to do is to link our original .HTML file to a new JavaScript file (.JS). Similar to the .CSS file, the .JS file will contain special code designed to manipulate or add to the content of the .HTML file.

The DOM

The Document Object Model (DOM) is used to view and interact with the HTML, CSS and JavaScript content. This is done directly in a web browser. As an example, you may want to change the size of a containers padding, this can be done in a visual way and in real time using the DOM.

Objects, Arrays And The Data They Contain

Both object and arrays are used to store data. Arrays are ordered or indexed. This means each element are assigned a number and ordered numerically starting from 0. Object on the other hand, Objects store data as "key-value pairs". This means they are stored and accessed by a unique identifier.

Functions

Functions are blocks of code that perform specific tasks. They can be reused and reffered to when necessary.
